# Longest streak

Write a function, longestStreak, that takes in the head of a linked list as an argument. The function should return the length of the longest consecutive streak of the same value within the list.

**test_00:**
```go
var a = Node{Val: 5}
var b = Node{Val: 5}
var c = Node{Val: 7}
var d = Node{Val: 7}
var e = Node{Val: 7}
var f = Node{Val: 6}

a.next = &b;
b.next = &c;
c.next = &d;
d.next = &e;
e.next = &f;

// 5 -> 5 -> 7 -> 7 -> 7 -> 6

longestStreak(a); // 3
```
